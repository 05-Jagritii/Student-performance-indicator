# Student Performance Predictor

A simple Machine Learning web application that predicts student exam scores based on study habits.

## Features

- Predict exam score based on:
  - Study hours
  - Attendance
  - Assignments completed
- Built with Linear Regression
- Interactive UI using Streamlit

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit

## Installation

Clone the repository:
```
git clone https://github.com/05-Jagritii/Student-performance-indicator.git
cd student-performance-predictor
```
Install dependencies:
```
pip install -r requirements.txt
```
Run the app:
```
streamlit run app.py
```


## Machine Learning Pipeline

1. Load dataset
2. Feature selection
3. Train-test split
4. Train Linear Regression model
5. Predict exam score
